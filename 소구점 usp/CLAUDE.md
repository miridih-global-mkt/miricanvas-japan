## 2. 칩스 표기 방식 — 호버 폐기, 항상 보임

기존 `data-src` 호버 툴팁 시스템 전면 제거. 다음 7개 클래스만 사용.

### CSS
```css
.chip{
  display:inline-block;
  padding:1px 7px;
  border-radius:3px;
  font-size:10px;
  font-weight:600;
  letter-spacing:-.005em;
  margin-left:6px;
  vertical-align:2px;
  border:1px solid;
  white-space:nowrap;
}
/* 플레이북 — 청록 (브랜드 톤) */
.chip.pb       {background:#E0FAFB;color:#02A8AD;border-color:#B2F0F2}
.chip.pb-seg   {background:#F0F5FF;color:#3B5BB8;border-color:#D6E1F5}
/* 경합 — 빨강 */
.chip.comp     {background:#FFF4F4;color:#B83A3A;border-color:#F5DDDD}
.chip.comp-aip {background:#FDEAEA;color:#9C2222;border-color:#F2C9C9}
/* 가이드 — 황금-갈색 */
.chip.guide    {background:#FBF4E4;color:#8B6914;border-color:#EDDFB8}
.chip.guide-d  {background:#F7EFD9;color:#6B4F0F;border-color:#E2CFA0}
/* 주요기능 — 자주 (실험·기능 분류) */
.chip.feat     {background:#FAF5FF;color:#6B21A8;border-color:#E9D5FF}
```

### 사용 규칙
- 섹션 헤더(`<h2>`) 또는 카드 헤더 옆에 한 번씩만
- 본문 카드마다 일일이 달지 말 것
- 복수 출처 = 칩스 2~3개 나란히

## 3. 콘텐츠 분류 기준 (작성자 머릿속에서만 사용)

미캔의 두 갈래 전략(AIP 기능 / 템플릿)은 **백서 구조 프레임이 아니라 작성자가 콘텐츠를 뒤섞지 않게 분류하는 머릿속 기준**.

| 분류 | 포함 |
|---|---|
| 🟦 AIP 쪽 | AIP 사용 흐름·STEP별 기능 전부 — STEP 1 입력 / STEP 2 채팅 / STEP 3 편집(1mm·Smart Block·시각화·요소검색·브랜드) / STEP 4 출력 |
| 🟪 템플릿 쪽 | 일본 전용 10만+ / 디자이너 퀄리티 |
| 🟨 공통/양쪽 | 가격·플랜 / 데이터 보안 / 일본어 AI 완성도 / 협업·공유 |
| 🟩 독립 부록 | **AI 기능** (이미지·아이콘·인포그래픽 생성, AI 라이팅, 배경/영역 제거 등) — AIP 후순위, 차트보다 우선 / **차트** — 장차 AIP로 통합 예정이나 현재 AIP에서 못 씀 |

> **백서에 "무기 ①/②" 같은 명시 라벨 절대 금지.** 분류는 머릿속에서만.

## 7. 작성·갱신 원칙

1. **호버 툴팁 폐기** — `data-src` 속성과 `[data-src]::after` CSS 사용 금지. 칩스만.
2. **두 무기 라벨 금지** — "🎨 무기 ②" 같은 명시 표기 절대 금지, 자연스러운 제목으로
3. **AIP/템플릿 콘텐츠 뒤섞기 금지** — 분류는 머릿속에서만 적용, 본문은 자연스럽게
4. **플레이북 공통 영역 디테일 우선** — STEP 시트 등 각각 충실히 풀어내기
5. **차트는 AIP 부속 아님** — 독립 부록. 현재 AIP에서 사용 불가임을 한 줄 명시
6. **세그먼트 페르소나·차트·AI 기능은 후반부 부록** — 본문 흐름과 분리. AI 기능 → 차트 → 페르소나 순.
7. **콘텐츠 가이드 PART 생성 금지** — 광고 투고 규칙·체크리스트·영상 스크립트 등은 백서에서 제외
8. **출처는 칩스 7종만 사용** — 플레이북 / 플레이북(세그별) / 경합(AIP) / 경합 / 가이드(개요) / 가이드(상세) / 주요기능
9. **약칭 사용 금지** — `PB-ALL`, `WIKI-1`, `INFLU-서비스` 같은 표기 금지. 칩스명만.
10. **언어** — 전체 한국어, **일본어 광고 카피·고객의 목소리(📣ユーザーの声)는 일본어 유지**
11. **★ 가이드(개요)/가이드(상세) 합성 의심 원칙** — 가이드 실무자(1년차)가 합성·요약한 정보일 수 있다. 분류·세그 구분·요약 데이터는 1차 출처(플레이북·경합·주요기능)에서 직접 확인 후 인용.
12. **★ 디테일 세그 분류 메인 위치 사용 금지** — 직군·업종·타깃 세그 분류는 캠페인·시즌별 변동. 백서 본문 최상단/메인 위치에 박지 말 것. 보조 예시 톤으로 후반부 배치.
13. **★ 출시 불확실·지역 한정·미확인 기능 인용 금지** — 5장 백서 인용 금지 표 참조.
