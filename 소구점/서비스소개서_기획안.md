## 1. 기획 방향

### 타깃 독자
일본 중견~대기업 마케팅/영업/HR/경영기획 담당자.  
결정권자에게 올리고 싶어지는 "독り歩きできる(혼자 걸어 다니는)" 자료를 만드는 것이 목표.

### 감정 흐름 설계 (アジコイネ × PASONA 통합)

```
[P]  문제:   こんな悩みありませんか？ → Yes가 나오는 상황 설정
[A]  위기감: このまま放置すると…   → 비용·리스크 수치로 가시화
[ア] 이상:   こうなれたら…         → 미래상 제시
[コ] 해결:   MiriCanvasなら3つの理由 → 효과 로직 (매직 넘버 3)
[イ] 증거:   導入実績 + デカイ セット
[O]  제안:   POC 무료 체험
[N]  한정:   POC枠・期間限定
[ネ] 행동:   더블 바인드 CTA
```

### 핵심 원칙 5가지

1. **매직 넘버 3** — 이유는 3개, 강조 포인트 3개 이내
2. **독り歩き** — 담당자가 혼자 읽어도 상사 설득 가능한 구조
3. **전략적 ×** — 비교표에 약점 1개 포함해 전체 신뢰도 확보
4. **デカイ** — 수치 + 얼굴사진 + 인터뷰 목소리 세트
5. **더블 바인드** — 어느 쪽 선택해도 '다음 단계'로 이어지는 CTA
