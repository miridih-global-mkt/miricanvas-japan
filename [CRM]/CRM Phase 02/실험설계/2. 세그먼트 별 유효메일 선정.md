## 보정 원칙 요약

| 캠페인 유형 | 처리 방식 |
|---|---|
| **JAS** (8통 시퀀스) | 시리즈 전체를 하나의 블록으로 등급 부여. 개별 cell 단위 선정 안 함 |

## 세그먼트별 정리

### NULL (n=655)

**JAS 블록 등급**: 8통 전부 v7 강양 + kpi≥2 → **A (확실한 효과)**
- 전 cell: v7 19~27, kpi 2~4. 가장 강력한 증거.

| 등급 | 항목 | v7 | KPI | 비고 |
|---|---|---|---|---|
| **A** | JAS 시리즈 전체 (8통) | 19~27 | 2~4 | 블록 전체 일관 A |
| C | D2 data_table | 7 | -1 | v7만 양, KPI 미달 |

| C | JA1 4step | 8 | 0 | v7만 양 |
| C | EN1 welcome | 5 | 0 | v7만 양, KPI 미측정 |

### STUDENT (n=260)

**JAS 블록 등급 (⚠ 이전 v0.1 오류 수정)**:
- 발송 순서 1~3번 (4step_empathy → aip_curiosity → font): v7 28~32, kpi=2 → A
- 발송 순서 4~8번 (bgremove → table → graph → templatemix → bookmark): v7 19~32, kpi=0 → C
- 결론: 시리즈 초반이 KPI 효과 주도. 블록 전체 = **B (정착 효과)**

> v0.1의 "JAS aip_curiosity / font / 4step_empathy = A 3개" 표기는 **시리즈 confound 미반영한 오류**. 블록으로 처리.

| 등급 | 항목 | v7 | KPI | 비고 |
|---|---|---|---|---|
| **A** | D2 data_table | 6 | 3 | 단발 A/B, 양쪽 모두 확인 |
| **B** | JAS 시리즈 전체 (8통) | 19~32 | 2 (초반) | 블록 등급. 초반 3통 KPI 확인 |
| C | JA1 review | 9 | 0 | v7만 양 |
| C | JA1 aip | 4 | 0 | v7만 양 |
| C | JA1 4step | 6 | 0 | v7만 양 |
| **E** | D2 image_bgremove | **-9** | 0 | v7 강한 역효과 (≤-7) ⚠ |
| **E** | D1 4step_empathy | **-10** | 0 | v7 강한 역효과 (≤-7) ⚠ |
