C. 좌측 라이브러리 패널 + 드래그

편집 화면 좌측에 라이브러리 패널 항상 노출. 드래그해서 원하는 위치에 떨어뜨림.
장점: 모든 종류가 한눈에 보임. 시각적
단점: 화면 폭 차지 (편집 영역 좁아짐). 비교 탭 등에선 안 쓰는데도 자리 차지. 드래그 정밀도 필요

💡 추천: A (인라인 + 버튼)
• 마우스 hover에 의존 X → 발견성 높음
• 클릭 → 팝오버에서 6/12개 중 검색·선택 (영문/한글 키워드 모두 검색)
• 추가 위치가 명확 (어디에 들어갈지 헷갈리지 않음)
• 화면 폭 보존 (라이브러리 패널처럼 자리 차지 X)

HTML 코드 표시 <textarea readonly> 코드 보기에서 복사용. 신택스 하이라이트는 v2로
