속도감 비교 (실제 시간 기준)

  • A (인라인 + 팝오버): + 클릭 (0.5초) → 카드 클릭 (0.5초) = 약 1초. 키보드 ↑↓+Enter도 가능
  • C (좌측 패널 드래그): 패널에서 항목 찾기 (0.5초) → 드래그해 정확한 위치에 떨어뜨리기 (1~2초) = 약 1.5~2.5초

실은 A가 더 빠를 가능성이 높습니다 — 드래그 정밀도가 의외로 시간 잡아먹어서. 그리고 화면 폭도 안 차지함.

대안: 하이브리드
"+ 섹션" 버튼을 길게 누르면(또는 우클릭하면) → 라이브러리 패널이 임시로 우측에 슬라이드 인 → 거기서 드래그도 가능. 평소엔 깔끔, 가끔은 시각적. 단, 구현 복잡도 약간 ↑
