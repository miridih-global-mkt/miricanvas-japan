1.1

MiriCanvas란 무엇인가

MiriCanvas(ミリキャンバス)는 한국 미리디(MIRI D.I.H)가 개발한 AI 기반 올인원 디자인·프레젠테이션 플랫폼. 텍스트 입력 한 줄 또는 PDF 업로드만으로 구성·디자인·편집이 완성된 슬라이드를 3분 안에 생성한다.

AI 프레젠테이션 (AIP)
텍스트·PDF 입력 → 목차·슬라이드 자동 생성. 생성 중에도 편집 가능. 30장·무료·워터마크 없음.
올인원 디자인 툴
슬라이드 외 포스터·SNS·전단·보고서·인포그래픽까지. 템플릿 63만+ / 소재 3,200만+.
일본 로컬 핏
일본 전용 템플릿 10만+ / 일본어 AI 생성 완성도 / 일본어 언어 필터. 글로벌 툴이 못 채우는 로컬 디테일.

타겟 사용자 3대 축

💻 비즈니스
영업·경영지원·마케팅. "시간은 없고 퀄리티는 포기 못하는" 직장인.
✏️ 교육
초중고 교사·학원강사·교육위원회. 수업 자료 반복 제작 부담 해소.
👩‍⚕️ 의료·간호
간호·의료 관계자. 연수·설명 자료를 빠르게 전문적으로.
核心메시지 (일본 유저 목소리): "AIっぽくなく、そのまま仕事で使えるデザイン" / "日本のビジネス資料にも違和感なく使える" / "デザイン知識がなくても短時間で完成" / "これ1つで完結するからラク"
황금 유저 인사이트 (플레이북)
"템플릿 생성 + 편집 경험을 함께 한 사람들이 미리캔버스의 황금 유저"
유의미하게 생성되는 템플릿 자체로도 자료의 80% 완성. 나머지 20%는 편한 편집 경험이 채운다. 두 경험을 모두 느낀 사람이 잔존율·NPS 기준 황금 유저로 정의됨. → 홍보 시 "생성 + 편집" 두 단계를 모두 체험하게 유도할 것.
1.2

よくある悩み — 4가지 공통 고민과 미캔의 답

일본 사용자 조사에서 반복 등장하는 4가지 핵심 고민. 파트너 콘텐츠의 공감 포인트로 직접 활용 가능하다.

① スライド作成に時間がかかる
슬라이드 만드는 데 시간이 너무 걸린다. → 미캔: 텍스트 한 줄 → 3분 이내 30장 완성. 구성부터 디자인까지 AI가 처리.
② AIっぽいデザインになってしまう (핵심 불안)
AI 도구를 쓰면 "AI티가 난다"는 불안. 실제 이탈 1위 원인. → 미캔: 일본 전용 10만+ 템플릿 기반 생성으로 "そのまま提出できるクオリティ".
③ 構成を考えるのが面倒
무엇을 어떻게 구성할지 결정하는 것 자체가 귀찮다. → 미캔: 생성 전 목차 채팅으로 방향 수정. AI가 목차·구조를 먼저 제안.
④ 本来業務に集中できない
자료 만드는 데 시간이 빼앗겨 본업에 집중 못한다. → 미캔: 자료 제작 시간 단축 → 본업 집중. "これ1つで完結するからラク."
콘텐츠 활용 팁: "AIっぽいデザインになってしまう" 고민은 일본 사용자 이탈의 핵심 원인. 홍보 콘텐츠에서 Before(AI티 나는 결과물) / After(미캔 결과물) 비교를 직접 보여주는 것이 가장 강력한 설득 방식.

STEP 1 — 입력 · 3분 초안 생성

"어떤 형태로 넣어도 AI가 목차와 슬라이드를 자동 설계한다." 3가지 입력 방식과 업계 최다 슬라이드 수가 핵심.

① 문장 한 줄 입력
짧은 주제·키워드만으로 목차 자동 생성. "주제만 던지면 구조가 잡힌다."
② PDF 업로드
기존 보고서·문서 → 슬라이드 즉시 변환. "보고서 올리면 프레젠테이션 완성."
③ 긴 글 최대 2만자
방대한 기획안·문서 통째로 처리. Canva·Gamma는 수천 자 제한.

2.2

STEP 2 — AI 아이데이션 · 채팅 수정

생성 전·후 모두 채팅으로 방향·구조·내용을 자유롭게 조정. 다른 AI 툴을 오가지 않아도 된다는 것이 핵심.

목차 단계 채팅 (생성 前)
목차 확인 후 "이 부분 더 자세히" 등 자유롭게 요청. "생성 전부터 내 방향대로."
생성 후 자연어 수정
"슬라이드 3에 타겟 추가해줘" — 말로 수정하면 AI가 반영.
내용 보강·추가
섹션 추가·논리 보강·표현 변경 채팅 한 줄로. "초안에서 내용 보충도 채팅 한 줄로."
경쟁사 (Canva)
채팅 아이데이션 없음. 생성 후 직접 편집만 가능.
MiriCanvas
생성 전 목차 채팅 + 생성 후 자연어 수정. 원스톱으로 완결.

세일즈 포인트: Gamma를 쓰다가 "편집이 안 돼서" 이탈한 사용자에게 가장 강력한 소구. "Gamma처럼 만들고 막히는 일 없음"이 핵심 카피.

STEP 4 — 출력 · 다운로드

일본 직장인의 90%가 PowerPoint를 사용한다. PPTX 완벽 호환 무료 출력은 일본 시장에서 결정적 소구점.

PPTX 완벽 호환 무료
레이아웃·문자 깨짐 없이 저장. "회사에서 파워포인트만 써도 예쁜 프레젠테이션 만들 수 있어요." Gamma: PPTX 불가 (치명적 약점)
워터마크 없음 무료
무료 플랜에서도 워터마크 없음. Gamma: 무료=워터마크 있음 / GenSpark: 워터마크 있음
서비스 로고 숨기기 무료
미캔 로고 없이 다운로드. Canva·Gamma: 유료
업계 최다 출력 형식
PNG(배경 투명 가능) · JPG(Web/인쇄용) · PDF(Web/인쇄용) · PPTX · MP4 · GIF — 모두 무료.
공유 링크
가입 없이 링크 하나로 즉시 공유. "링크 하나로 즉시 공유."
데이터 보안
제3자 미제공 · AI 재학습 없음 명시 (예정). "회사 자료 넣어도 안전."

⚠ 약점

파트너가 알아야 할 미캔의 약점 (반박 대응용)

경쟁 비교 시 사용자가 지적할 수 있는 미캔의 실제 약점. 사전에 인지하고 반박 논리를 준비해둘 것.

약점 항목실제 상황반박 / 보완 논리
첨부파일(파일 첨부) 유료 슬라이드 내 파일 첨부는 유료 플랜에서만 가능. Canva·Gamma는 무료. PPTX 완벽 호환 + 무료 다운로드로 대체. "첨부파일 대신 PPTX로 보내면 동일하게 공유 가능." 핵심 AI 기능(생성·편집·배경제거·출력)은 전부 무료.
Smart Block 되돌리기 일부 불가 스마트블록 내 특정 조작은 Ctrl+Z(되돌리기)가 적용되지 않는 경우 있음. 버전 이력 기능(Pro)으로 이전 상태 복구 가능. 스마트블록은 언제든 "깨기"로 해제하여 자유 편집 전환 가능.
파트너 행동 수칙: 약점을 모른 척하지 말 것. "이런 점은 아직 아쉽지만, 대신 이렇게 해결하면 된다"는 솔직한 안내가 신뢰도를 높인다. 위 반박 논리를 숙지 후 콘텐츠 댓글·DM 대응에 활용.

STEP별 소비자 언어 (카피방향 — 플레이북 원문)

STEP 1 — 입력 단계
"주제만 던지면 구조가 잡힘" / "보고서 올리면 프레젠테이션 완성" / "방대한 기획안도 통째로" / "원하는 분량 맞춤 설정"
어떤 형태로 입력해도 AI가 목차와 슬라이드 구조를 자동 설계 — 핵심 소구는 "속도"와 "간편함"
STEP 2 — AI 아이데이션 단계
"생성 전부터 내 방향대로" / "말로 수정하면 AI가 반영" / "초안에서 내용 보충도 채팅 한 줄로" / "目次の段階から方向を調整できる、そのままビジネスに使えるAI。"
생성 전·후 모두 채팅 조정 가능 — Canva·Gamma·GenSpark에 없는 단독 소구 포인트
STEP 3 — 에디터 편집 단계
"긴 글 넣으면 표·차트가 데이터로 바로" / "Gamma처럼 만들고 막히는 일 없음" / "원하는 스타일 요소 검색하면 바로" / "내용 고쳐도 레이아웃 안 무너짐" / "표 색상·차트 두께도 내 마음대로" / "브랜드 컬러 그대로 적용"
생성 후 막히는 일이 없음 — Gamma·GenSpark의 편집 약점을 정면으로 해결
STEP 4 — 출력 단계
"회사에서 파워포인트만 써도 예쁜 프레젠테이션 만들 수 있어요" / "링크 하나로 즉시 공유" / "회사 자료 넣어도 안전"
일본 직장인 90%가 PPT 사용 — 기존 업무 방식을 바꾸지 않아도 된다는 약속
