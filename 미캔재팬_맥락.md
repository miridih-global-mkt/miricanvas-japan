# 미리캔버스 재팬 — 종합 상황 맥락 (2026.05)

## 서비스 개요
미리캔버스(MiriCanvas / ミリキャンバス)는 한국 미리디(MIRI D.I.H)가 개발한
AI 기반 올인원 디자인·프레젠테이션 플랫폼.
텍스트 입력 한 줄 또는 PDF 업로드만으로 구성·디자인·편집이 완성된
슬라이드를 3분 안에 생성한다.

---

## 현재지표
- MAU(액티브 기준: 로그인) 3만 명
- MAU 중 retain 유저 비율 20%
- 주간 마케팅비 5,000만 원
- 주간 방문자: 페이드 4만 명 / 오가닉 2만 명
- 주간 가입자: 페이드 3,000명 / 오가닉 3,000명
- 가입 전환율 10~15%
- 페이드+오가닉 UAC 6,000~7,000원
- W1 리텐션 5~6%
- M1 리텐션 4~5%
- 유료 유저 1,240명

  이루시루(Irusiru), 톰(Tom), 스카이워크, 펠로 등 신생 경쟁사 다수
- 대체로 검색광고 위주 집행 중
- 젠스파크는 TV광고부터 메타 DA까지 전방위적 IMC 마케팅 공세 중

  → 이들의 AIP 기능은 기획·웹서치 등 앞단 과정부터 PPT 출력까지 전과정 처리

- 대다수 일반 대중은 에이전트형 AI의 존재 자체를 모름
  (예: 한국의 정육점 사장님이 노션을 모르는 것과 유사한 상황)
- 일본 직장인 90%가 PowerPoint 사용 중인 환경

### 잠재 사용자 특성
- 보통 30대 이상의 특정 부서 담당자가 PPT를 반복적으로 많이 만듦
- 얕은 실력의 웹마케팅, 개발, 디자인 계열 컨설팅/부업을 하는 3040 여성 다수
- 리스킬링을 원하는 사람들 많음

## 자사 강점

### 제품 포지셔닝
- 캔바(Canva)보다 훨씬 니치한 타겟/상황을 위한 템플릿 다수 보유
- 캔바보다 템플릿 퀄리티 압도적으로 높음
  (모든 템플릿을 프로 디자이너가 직접 기획·제작·검수)
- 캔바보다 AI 이미지/영상 생성 기능이 훨씬 다양함
- **AI 프레젠테이션(AIP)이 최대 강점**

### AIP의 차별점
- 다른 에이전트형 AI(젠스파크, 감마 등)와 달리:
  - 초심자 / 디자인 비전문가 / IT 리터러시가 낮은 사람이 쓰기 좋음
  - 프레젠테이션뿐 아니라 포스터·SNS 콘텐츠·전단·보고서·인포그래픽 등
    다양한 디자인 작업을 하나의 툴로 완결
- 가입 유저 획득 비용은 다소 높지만 리텐션도 높고 유료 구독 전환율도 가장 유의미함

### 일본 로컬 특화
- 일본 전용 템플릿 10만+ (글로벌 툴에는 없는 영역)
- 일본어 AI 생성 품질 우수 (이상한 일본어 표현 없음)
- 일본어 검색 시 일본 템플릿만 노출되는 언어 필터

## AIP 4단계 기능

```
STEP1 입력·3분 초안 → STEP2 AI 아이데이션 → STEP3 에디터 편집 → STEP4 출력
```

| 단계 | 핵심 기능 | 경쟁사 대비 우위 |
|------|-----------|----------------|
| STEP 1 입력 | 문장 / PDF / 긴 글 2만자 / 6~30장 지정 | Canva는 문장만, Gamma PDF 가능하지만 20장 제한 |
| STEP 2 채팅 | 생성 전 목차 조정 + 생성 후 자연어 수정 | Canva 없음, Gamma 생성 후만 가능 |
| STEP 3 편집 | 1mm 완전 편집 + Smart Block + 시각화 자동 변환 | Gamma 폰트·위치 변경 불가 (치명적 약점) |
| STEP 4 출력 | PPTX 완벽 호환 · 무료 · 워터마크 없음 | Gamma PPTX 불가, GenSpark 유료만 |

## 타겟

### 메인 타겟
- 비즈니스 직장인 (영업, 광고/마케팅, 기획/사업부, 인사)
- 교사·강사·연수 담당자

### 검토 중인 후보 타겟
- 부업 종사자
- 프리랜서
- 스타트업

### 타겟 제외
- 대학생/대학원생 (발표·세미나·취준)
- 디자이너·일러스트레이터·콘텐츠 크리에이터

### 황금 유저 인사이트
> "템플릿 생성 + 편집 경험을 함께 한 사람들이 미리캔버스의 황금 유저"
> → 생성만 해보고 끝나는 게 아니라, 편집까지 해본 사람이 잔존율·NPS 기준
>   황금 유저로 정의됨. 홍보 시 "생성 + 편집" 두 단계를 모두 체험하도록 유도.
