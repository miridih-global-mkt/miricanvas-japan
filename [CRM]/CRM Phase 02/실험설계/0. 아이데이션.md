## 1. 합의된 원칙

### 1.1 audience
- **국가**: 일본 한정 (`country = 'JP'` 기준, ISO 코드)
  - `language = 'ja'` 대신 `country = 'JP'` 채택 — 언어 흔들림보다 깨끗
- **segment 분기**: 7개 모두 단독 트랙 운영
  - NULL, STUDENT, COMPANY, INDIVIDUAL, EDUCATION, BUSINESS, INSTITUTION
  - INSTITUTION(29명) 포함 — evergreen 운영 시 모수 누적

### 1.2 운영 형태
- **evergreen 온보딩 시퀀스** (가입 직후 트리거)
- **이번 실험 운영 기간**: ~1달
- **arm fixed**: 한 번 배정되면 같은 시퀀스 유지 (검정 깔끔)
- **1달 후 결과 리뷰**: 명백한 winner면 다음 cohort부터 비중 조정

### 1.3 cell 선정 룰 — 3-tier 포함 + 2-tier 제외

| Tier | 정의 | 운영 |
|---|---|---|
| **A 검증** | KPI 양 (composite ≥ 2) AND v7 양 (composite ≥ 3) | 메인 (자신감 ⭐⭐⭐) |
| **B KPI 신뢰** | KPI 양 only | 메인 (자신감 ⭐⭐, 안착 직접 측정) |
| **C v7 실험** | v7 양 only | 실험적 포함 (자신감 ⭐) |
| **D 제외** | 양쪽 음 (KPI ≤ -2 AND v7 ≤ -3) | 발송 X |
| **E 제외 검토** | 한쪽이 강한 음 (KPI ≤ -4 OR v7 ≤ -7) | 다른 쪽 양이어도 검토 |

> cross-reference는 게이트 아닌 **ranking**. 양쪽 검증이 가장 신뢰가지만 한쪽만이라도 신호 있으면 후보.

## 3. 결정 필요한 열린 질문

### 3.1 JAS 시퀀스 확장 여부 (☐)
이전 시즌 JAS는 NULL·STUDENT·COMPANY·INDIVIDUAL에만 발송됨. EDUCATION·BUSINESS·INSTITUTION엔 측정 없음.

- **(보수)** 이들에 JAS 안 보내고 기존에 발송된 cell만 — 검증 안 된 것 안 보냄
- **(확장)** NULL·STUDENT에서 잘 됐던 JAS를 이들에게도 발송 — 신호 있으면 다음 시즌 확장 근거

### 3.2 arm 수 / 시퀀스 통수 (☐)
- NULL은 JAS 8통 전부 보낼지, 줄일지
- 작은 segment(BUSINESS/INSTITUTION)는 통수를 줄여서 신호 집중할지

### 3.3 holdout 비율 (☐)
- 전 segment 일괄 20%?
- 작은 segment는 더 낮춰서 운영 모수 확보?

### 3.4 발송 schedule (☐)
- D+1, D+3, D+5… 식의 간격 (이전 시즌 동일?)
- 메일 통수에 따라 조정 필요

### 3.5 E tier 제외 검토 (☐)
- 한쪽이 강한 음인 cell 검토 (현 데이터에선 명확히 잡힌 건 D1 aip_curiosity NULL 정도)
