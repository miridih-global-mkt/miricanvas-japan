### 1-3. 기능별 카피 뱅크

#### AIP (AI 프레젠테이션)
```
입력 방식:
- 「テキスト1行で、30枚のスライドが3分で完成します」
- 「PDFをそのままアップするだけ。スライドに変換します」
- 「最大2万文字まで。長い文書も一発でプレゼンに」

생성 단계 장점:
- 「目次の段階でチャットで調整できます」（Canva·Gamma 없음）
- 「生成後も自然言語で修正できます」

편집 장점:
- 「1mm単位で自由に配置。Gammaと根本的に違います」
- 「Smart Blockが内容変更後のレイアウト崩れを防ぎます」
- 「内容をそのまま表・グラフに自動変換（5月〜）」

출력 장점:
- 「PPTX完璧対応・無料・透かしなし。Gammaにはできません」
- 「PNG/JPG/PDF/MP4/GIF、すべて無料」
```

#### 템플릿
```
- 「日本専用テンプレート10万枚以上。海外ツールの翻訳版じゃありません」
- 「デザインの知識がなくても、プロ品質の資料が完成します」
- 「ブランドキットを登録すると、誰が作っても同じクオリティに」
```

#### 가격/무료
```
- 「まず無料で試せます。クレジットカード不要」
- 「無料プランでも、透かしなし・PPTX出力・AI生成30枚」
- 「Proは月980円。Canvaより200円安く、機能は上回ります」
```

### 1-4. 경쟁사별 전환 메시지

#### Gamma 유저에게
```
核心: "Gammaで作れるけど、編集できない" 경험을 가진 유저

- 「Gammaで作ったスライド、PowerPointで開いたら崩れませんでしたか？」
- 「MiriCanvasなら、生成後も1mmレベルで自由に編集できます」
- 「PPTX完璧対応。Gammaが唯一できないこと、私たちはできます」
- 「フォント変更、位置調整、完全自由。Gammaとの最大の違いです」
```

#### Canva 유저에게
```
核心: "Canvaは日本語が弱い, 背景除去が有料" 불만

- 「日本語でテンプレートを検索しても、日本のデザインだけが出てきます」
- 「背景除去、MiriCanvasなら無料。Canvaは有料です」
- 「日本専用テンプレートが10万枚以上。Canvaの翻訳版とは違います」
- 「AIスライドなら、MiriCanvas 30枚 vs Canva 20枚。同じ無料でも差があります」
```

#### Irusiru(이루시루) 유저에게
```
- 「MiriCanvasなら同時編集が可能です。Iruシルにはできません」
- 「AIプレゼンテーション機能、Iruシルにはありません」
```

```
CTA: 「提案書テンプレートを試す」
```

#### 마케팅 담당자

```
✅ 「ぜひお試しください」

```
