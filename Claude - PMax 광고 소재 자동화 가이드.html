도구 사용법

① Claude Code (터미널)

AI가 직접 Figma에 명령을 내리는 도구예요. 터미널에서 말로 지시하면 자동으로 실행합니다.
시작 방법
bash
# 1. 프로젝트 폴더로 이동 cd pmax-automation # 2. Claude Code 실행 (승인 없이 자동 실행 모드) claude --dangerously-skip-permissions

② Talk to Figma (플러그인)

Claude Code가 Figma와 통신하는 다리 역할이에요. 작업 전 항상 플러그인이 켜져 있어야 합니다.
Figma 열기 → 상단 메뉴 → 플러그인 → Talk to Figma 실행
플러그인 패널에 Connected 표시 확인
Claude Code에서 채널명 입력해서 연결
플러그인이 꺼져 있으면 Claude Code 명령이 Figma에 반영되지 않아요.

③ 프로젝트 폴더 구조

pmax-automation/ ├── .env ← Figma 토큰 (외부 공유 절대 금지!) └── CLAUDE.md ← Claude 행동 지침

카피만 변경할 때

{프레임명}의 headline-text 레이어를 찾아서 텍스트를 "{카피}"로 변경해줘. 텍스트가 {N}자를 초과하면 {N}번째 글자 뒤에 줄바꿈(\n)을 삽입해줘.
예시
JP_vs_sq_BASE_v1의 headline-text 레이어를 찾아서 텍스트를 "3分で完成するAIスライド"로 변경해줘. 텍스트가 7자를 초과하면 7번째 글자 뒤에 줄바꿈(\n)을 삽입해줘.

배경색만 변경할 때

{프레임명}의 bg-rectangle 레이어를 찾아서 fill 색상을 {hex코드}로 변경해줘.
예시
JP_vs_sq_BASE_v3의 bg-rectangle 레이어를 찾아서 fill 색상을 #7B61FF로 변경해줘.

글자 축소(shrink) 적용할 때

{프레임명}의 headline-text 레이어를 찾아서 텍스트를 "{카피}"로 변경해줘. 텍스트가 {N}자를 초과하면 font-size를 {size}로 줄여줘.
예시
JP_vs_land_BASE_v2의 headline-text 레이어를 찾아서 텍스트를 "日本語特化のAIスライド"로 변경해줘. 텍스트가 13자를 초과하면 font-size를 36으로 줄여줘.
