## 제작 의뢰: 일본 B2B 기업 대상 서비스 소개서 (프레젠테이션 슬라이드)

### 배경 및 목적
- **클라이언트**: MiriCanvas Japan (미리디 주식회사의 일본 서비스)
- **파트너사**: COA (일본 현지 리셀러)
- **용도**: COA가 기존 고객사를 대상으로 MiriCanvas POC 도입을 제안할 때 사용하는 영업자료
- **사용 장면**:
  - 고객사에서 문의가 들어왔을 때 전달
  - COA 영업 담당자가 직접 배포
  - 대면 제안 시 함께 제시
- **핵심 전제**: 기업 담당자(마케팅·영업·HR·경영기획)가 보는 B2B 자료. 담당자가 읽고 나서 상사에게 자발적으로 올리고 싶어지는 구조로 만들 것.

---

### 디자인 방향
- **톤**: 신뢰감 있고 깔끔한 일본 B2B 스타일. 과하지 않고, 정보가 읽히는 디자인.
- **언어**: 일본어 메인 (슬라이드 텍스트 전체 일본어). 회사명·브랜드명은 영문 유지.
- **컬러 팔레트**:
  - 메인: `#03D4DA` (MiriCanvas 시그니처 청록)
  - 다크: `#1E2A4A` (네이비, 헤드라인·강조)
  - 서브 강조: `#6B4FBB` (보라, 카피 포인트)
  - 배경: `#FFFFFF` / `#F7FAFA`
  - 성과 수치: `#E8A020` (골드)
- **폰트**: 일본어는 Noto Sans JP 또는 Hiragino Sans. 헤드라인 굵게, 본문 Regular.
- **슬라이드 비율**: 16:9
- **총 슬라이드 수**: 13장 (표지 포함)
- **내보내기 형식**: PPTX

#### 슬라이드 0 — 표지

**목적**: 5초 안에 "우리 회사도 이렇게 될 수 있겠다"는 이상 상태를 전달.
제품 기능이 아닌 **고객이 얻게 될 변화**를 메인 카피로.

**메인 카피** (크게, 네이비+보라 강조):
```
チームの誰もが、
プロクオリティの資料を当日中に。
```

**서브 카피** (작게):
```
テンプレート × AI で、営業・マーケ・HR が
それぞれ3分で本格デザイン
```

**로고**: MiriCanvas 로고 (좌상단 또는 중앙 하단) + COA 로고 병기
**배경**: 네이비 다크 또는 청록 그라디언트. 실제 자료 화면 목업 이미지 우측에 배치 (있으면).
**하단 소자**: `© MiriCanvas Japan × COA | 企業向けPOC導入ガイド`

#### 슬라이드 1 — 왜 지금인가? (신규)

**목적**: PASONA의 Agitation. "지금 행동하지 않으면 뒤처진다"는 긴박성 부여.
시장 데이터로 경쟁 압박감을 1장에 전달.

**헤드라인**:
```
AI活用の競争、すでに始まっています。
```

**본문 레이아웃**: 수치 3개를 크게 강조하는 stat 카드 형식

| 수치 | 설명 |
|---|---|
| 【n】% | 日本企業のうち、業務でAIツールを活用している割合（2025年） |
| 【n】時間 | 営業担当者が週に費やす資料作成の平均時間 |
| 【n】万円 | 1件の提案書を外注した場合の平均コスト |

**하단 메시지** (임팩트 있게):
```
競合他社はすでに動き出しています。
「検討中」のままでは、差がつくのは資料の質だけではありません。
```

**디자인**: 다크 배경(네이비) + 수치는 골드/청록 강조. 긴박감 있는 무드.

#### 슬라이드 2 — 직무별 고민 × 이상 상태

**목적**: "우리 얘기네!"라고 느끼게 하는 공감 슬라이드. 4직무의 현재 고통과 이상 상태를 Before/After로 전달.

**헤드라인**:
```
「あるある」と感じたら、それがMiriCanvasで解決できます。
```

**레이아웃**: 4개 직무 카드 (2×2 그리드). 각 카드 상단에 직무명 아이콘, 하단에 BEFORE(빨간 배경)→AFTER(청록 배경)

**카드 1 — 마케팅 담당**
- BEFORE: 外注依頼・修正繰り返しで工数がかかる / ブランドがバラバラ / トレンド対応が遅れる
- AFTER: ブランドキットで誰でも統一デザインを即日制作 / 外注コストを大幅削減
- KPI 수치: `制作リードタイム【n】日 → 当日`

**카드 2 — 영업 담당**
- BEFORE: 提案書を毎回ゼロから作成し2〜3日かかる / クオリティにばらつき / 1件あたり10〜20万円の外注費
- AFTER: 共有テンプレート＋AIで初稿を3分で完成 / 全営業のクオリティを統一
- KPI 수치: `提案書作成 平均【n】時間 → 3分`

**카드 3 — 경영기획 담당**
- BEFORE: 報告書・事業計画のたびにデザイナーへ依頼しスケジュールが遅れる / チームごとにフォーマットがバラバラ
- AFTER: テンプレートカスタマイズでチーム標準フォーマット確立 / データ可視化で説得力UP
- KPI 수치: `報告資料の工数【n】%削減`

**카드 4 — HR 담당**
- BEFORE: 採用広告を毎回ゼロから作成し1件数時間かかる / オンボーディング資料のクオリティが低い
- AFTER: 採用公告テンプレートで1時間以内に完成 / プレミアムテンプレートで研修資料の信頼感UP
- KPI 수치: `採用広告作成 【n】時間 → 1時間以内`

#### 슬라이드 3 — MiriCanvasが選ばれる2つの理由

**목적**: コ (효과의 로직). 기능→행동변화→성과의 인과관계를 명확히 제시.

**헤드라인**:
```
MiriCanvasが選ばれる、2つの強み
```

**레이아웃**: 좌우 2분할. 각각 말풍선 + 기능 아이콘 카드 형식.

**왼쪽 — 디자인 도구로서의 강점**
- 타이틀: `①テンプレートの圧倒的な量と質`
- 본문: 日本向けビジネステンプレート【n】万点以上 / SNS・プレゼン・チラシ・報告書まで網羅 / デザイナー品質を誰でも再現
- 키워드 태그: `52万点のテンプレート` / `日本語対応` / `ブランドキット`

**오른쪽 — AI 자료 작성 도구로서의 강점**
- 타이틀: `②AI資料作成で、初稿3分`
- 본문: テキスト・PDF・長文を入力するだけでAIが資料を自動生成 / スライド枚数は6〜30枚を選択可能 / 生成後もAIチャットで自由に編集
- 키워드 태그: `AIスライド無制限` / `AIライティング` / `AI画像生成`

**하단 로직 플로우** (작은 사이즈):
```
入力（3分）→ AI初稿生成 → チャットで修正 → 1mm単位で編集 → 即日出力
```

#### 슬라이드 5 — 国内導入実績

**목적**: 사회적 증명. 숫자와 로고로 "다들 쓰고 있다"는 신뢰 생성.

**헤드라인**:
```
すでに3,000社以上が選んでいます
```

**레이아웃 상단**: 대형 수치 3개 나열
- `3,000社以上` — 累計導入企業数
- `【n】万人` — 日本ユーザー数
- `【n】点` — 平均レビュースコア (★【n】/5)

**레이아웃 하단**: 대표 도입 기업 로고 띠 (한국 주요 고객사 로고 6~8개)
- 로고 아래: `※韓国国内実績。日本での展開を加速中。`

**서브 카피**:
```
韓国最大のデザインプラットフォームとして実績を積み、
日本市場へ本格展開。
```

#### 슬라이드 6 — 日本導入事例 (데카이 원칙)

**목적**: イ (인터뷰 사례). 데이터+얼굴 사진+인터뷰 텍스트 3세트로 진짜 신뢰 생성.
가장 공을 들여야 하는 슬라이드.

**헤드라인**:
```
日本でも、実際に変わりました。
```

**레이아웃**: 사례 2건을 좌우 또는 상하로 배치. 각 사례 카드에 아래 요소 포함.

**사례 1 — 大塚食品（식품 제조업 / DX 추진팀）**
- 회사명·업종 배지
- 도입 담당자 사진 자리 (`【담당자 사진 삽입 필요】`)
- 도입 전 과제: `提案資料の作成に1件あたり【n】時間かかっており、対応スピードに課題があった`
- 도입 후 성과 수치 (크게): `工数を【n】分の1に削減`
- 담당자 인터뷰 텍스트 (말풍선 형식):
  ```
  「【担当者コメント挿入】
  — 大塚食品 DX推進担当 【氏名】様」
  ```
- 주의: 수치 검증 필요 (1/3 삭감 기준 확인 要)

**사례 2 — リコレ（EC 사업자）**
- 위와 동일 구조
- 도입 전 과제: `【n】`
- 도입 후 성과: `追加費用0円でデザイン内製化を実現`
- 담당자 코멘트: `【リコレ担当者コメント挿入】`

**하단 작은 텍스트**:
```
※ 導入初期には【n】という課題もありましたが、
  サポートチームとともに解決し、現在は安定稼働中です。
```
(실패 극복 스토리 — 너무 완벽한 사례만 나열하면 오히려 의심받음)

### Claude Design 추가 지시사항

1. **각 슬라이드는 1 메시지, 1 액션**으로. 정보 과부하 금지.
2. **수치는 크게, 텍스트는 작게**. 의사결정자가 5초 안에 핵심을 파악할 수 있어야 함.
3. **Slide 6 (도입 사례)**가 가장 중요한 슬라이드. 디자인 공수를 가장 많이 투입.
4. **경쟁사 비교표 (Slide 7)**의 전략적 ×는 반드시 포함. 전부 ○이면 오히려 신뢰 저하.
5. **CTA (Slide 8, 12)**는 QR코드 자리를 반드시 확보.
6. **일본어 카피**는 자연스러운 일본 B2B 문체로. 직역체 금지.
7. PPTX 내보내기 후 PowerPoint에서 추가 편집 가능한 구조로 만들 것.
