카탈로그의 표현 새 라이브러리에서
Hero (로고 + H1)1열 섹션 + 이미지 모듈 + 텍스트 모듈(H1)
Body (강조박스 + 본문)1열 섹션(bg light teal) + 텍스트 모듈들
card_2col_grid2열 50/50 섹션 + 각 칸에 이미지+텍스트+버튼
cta_microcopy / cta_body_microcopy1열 섹션 + 텍스트(소형) + 버튼
Section Header (구분 띠)1열 섹션(bg light teal, 패딩 작게) + 텍스트(H2)
Box (사이드 토픽 박스)1열 섹션(bg light teal) + 텍스트 + (옵션) 이미지·버튼
핵심 통찰: 카탈로그의 13개 "모듈"은 사실 섹션과 모듈 조합 패턴이었음. 새 정의는 그 조합을 사용자가 직접 쌓도록 함 → 무한한 가짓수 가능. 자주 쓰는 조합은 v2에서 "프리셋"으로 따로 묶을 수도.
