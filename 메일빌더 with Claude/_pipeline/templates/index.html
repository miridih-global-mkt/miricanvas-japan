새 캠페인 만드는 법

Claude Code 채팅창에 자유롭게 말하세요. 구조는 안 맞춰도 됩니다.

예) 새 캠페인 시작: 타겟 = "가입 후 7일차 사용자", 주제 = "폰트 바꾸기"
예) B2B 구독 전환 유도 메일 하나 짜줘, 무료 체험 종료 알림 겸해서
예) 지난달 워크샵 참가자한테 감사 메일 보내고 싶어, 가볍고 따뜻한 톤으로
